> Hands-on NLP experiments and projects by **Bhargava Avancha** — AI/ML Engineer | Data Scientist | Conversational AI Enthusiast.

## Overview

This repository contains practical NLP experiments covering text preprocessing, sentiment analysis, text classification, and more using Python and Jupyter Notebooks.

## Projects

| Notebook | Description |
|----------|-------------|
| `NLP_practice.ipynb` | Core NLP practice covering tokenization, stemming, lemmatization, and bag-of-words |
| `Twitter Sentiment Analysis.ipynb` | Sentiment analysis on Twitter data using Naive Bayes classifier |

## Techniques Covered

- Text Preprocessing (Tokenization, Stop-word removal, Stemming, Lemmatization)
- Bag of Words (BoW) & TF-IDF
- Naive Bayes Classifier for Sentiment Analysis
- Model Serialization with Pickle
- Sentiment Analysis on Social Media Data

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| NLTK | NLP preprocessing |
| Scikit-learn | Machine learning models |
| Pandas / NumPy | Data manipulation |
| Jupyter Notebook | Interactive development |
| Matplotlib | Visualization |

## Getting Started

```bash
git clone https://github.com/Avancha-Bhargava/NLP.git
cd NLP
pip install nltk scikit-learn pandas numpy matplotlib jupyter
jupyter notebook
```
